## 프로젝트 소개
자바기반응용프로그래밍 기말과제, 카톡과 슬랙과 같은 채팅 프로그램을 만들어보며 소켓프로그래밍과 멀티스레드를 활용해보고자 함

## 기능
- 회원가입/로그인
- 채팅방 생성/참여
- 메시지 전송

## 설치 및 실행 방법
- JAVA 21이상 설치
- 프로젝트 클론하기
- ProgramServer 실행 후, Router 실행
